Q.3 Semantic element in HTML5 ?

Ans:-

  • Semantic HTML elements are those that clearly describe their meaning in a human- and machine-readable way. Elements such as header , footer and article are all considered semantic because they accurately describe the purpose of the element and the type of content that is inside them

Q.4 Canvas and SVG tags ?

Ans:-

  • SVG and Canvas are HTML5 APIs for rendering vector and raster graphics, respectively. SVG is used to create vector-based graphics,
    whereas Canvas can render both vector and raster graphics. Canvas is better for quickly rendering graphics and animations with less control than SVG.
